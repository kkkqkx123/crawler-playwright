# 豆瓣滑块验证码自动登录

这是一个使用Playwright和OpenCV自动解决豆瓣滑块验证码并登录的项目。

## 功能

1. 自动访问豆瓣登录页面
2. 点击密码登录控件
3. 输入用户名和密码
4. 自动识别并解决滑块验证码
5. 完成登录流程

## 安装依赖

```bash
pip install -r requirements.txt
playwright install
```

## 配置

1. 复制`.env.example`文件为`.env`（如果存在）
2. 在`.env`文件中设置你的豆瓣账号：

```env
DOUBAN_USERNAME=你的豆瓣用户名或邮箱
DOUBAN_PASSWORD=你的豆瓣密码
```

## 使用方法

运行主程序：

```bash
python main.py
```

程序将自动打开浏览器，执行登录流程，并尝试解决滑块验证码。

## 工作原理

1. **页面导航**：访问豆瓣登录页面
2. **表单填写**：
   - 点击密码登录标签
   - 输入用户名
   - 输入密码
   - 点击登录按钮
3. **验证码识别**：
   - 使用OpenCV模板匹配识别滑块缺口位置
   - 计算需要滑动的距离
4. **模拟拖动**：
   - 生成人类行为的拖动轨迹（加速、减速、随机抖动）
   - 使用Playwright模拟鼠标拖动操作
5. **结果验证**：检查是否登录成功

## 文件结构

- `main.py`：主程序文件
- `requirements.txt`：Python依赖包
- `.env`：环境变量配置文件（包含用户名和密码）
- `workflow.md`：豆瓣登录页面元素选择器文档

## 注意事项

1. 程序使用有头浏览器模式，以便观察登录过程
2. 滑块验证码识别可能需要多次尝试
3. 豆瓣可能会更新页面结构，需要相应更新选择器
4. 请勿频繁使用，以免触发反爬虫机制

## 故障排除

1. **找不到元素**：检查`workflow.md`中的选择器是否仍然有效
2. **验证码识别失败**：尝试增加`max_retries`参数
3. **登录失败**：检查用户名和密码是否正确，或豆瓣是否有其他验证机制
